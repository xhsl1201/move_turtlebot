시스템 명세서

1. 개요
본 시스템은 ROS2 기반으로 하며 PySide6 GUI를 활용하여 Turtlebot3의 원격 수동 주행과 자율 안전 로직을
통합 제어하는 컨트롤러로 구현

3. 주요 기능 
 - GUI를 통한 전진, 후진, 좌회전, 우회전 및 정지 명령을 입력할 수 있으며, 
   해당 명령은 /cmd_vel 토픽으로 퍼블리시되어 로봇의 수동 주행을 제어

 - 로봇은 /scan 토픽의 라이다(LiDAR) 센서 데이터를 구독하여 주변 장애물을 탐지하며
   안전 모드가 활성화된 상태에서 일정 거리 이내의 장애물이 감지되면 로봇의 주행 정지하도록 설계

 - 안전 모드는 SetBool 서비스(/set_safety_mode)를 통해 ON/OFF 제어가 가능하며, 
   ON 상태에서는 충돌 방지 기능이 동작하고 OFF 상태에서는 수동 제어가 우선되도록 구현

 - 장애물이 감지되면 액션(/turtlebot3 Patrol Action)을 사용하여 로봇이 제자리 회전을 수행하도록 하였으며
   또한 GUI의 STOP 버튼을 통해 액션을 취소할 수 있도록 구현하여, 회전 동작 중에도 사용자가 즉시 제어권 회수 가능

3. 시스템 통신 요약
   본 시스템은 토픽(수동 제어), 서비스(안전 모드 제어), 액션(장애물 감지 후 회전)을 역할에 맞게 분리하여 
   ROS2 통신 구조에 적합한 제어 시스템으로 구성
